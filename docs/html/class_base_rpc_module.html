◆ $timerDef

array BaseRpcModule::$timerDef =['ONLINE_INTERVAL'=>[0,'s']
protected

Setup the Timers Override this to configure Timers for Module.

ONLINE_INTERVAL checks when device is Online * OFFLINE_INTERVAL checks when device is offline

Parameters for Checks are [0,'s'] first are the minimum value 0 = timer disabled and second are the multiplicator s = seconds, m = minutes and h are hours

Note
Override in your Module to configure Timers
